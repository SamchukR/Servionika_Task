Хранилище ключ значение

Прототип API хранилища ключ-значение

Получить все данные хранилища WEB

http://127.0.0.1:5000/api/v1/storage/json/all

Получить все данные хранилища CURL

curl -i -X GET http://{127.0.0.1:5000}/api/v1/storage/json/all

Получить данные хранилища WEB

http://{127.0.0.1:5000}/api/v1/storage/json?key=test

Получить данные хранилища CURL

curl -i -GET http://{127.0.0.1:5000}/api/v1/storage/json/read?key=test

Добавить данные в хранилище хранилища WEB

curl -i -H "Content-Type: application/json" -X POST -d '{"test3": "value4"}' http://{127.0.0.1:5000}/api/v1/storage/json/writе
